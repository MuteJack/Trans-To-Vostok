## How to add yourself

If you would like to contribute, please submit a pull request that
includes your contribution **and** a corresponding update to this file
under the appropriate section (Translators / Contributors).

Suggested entry format:

```
- **Name (or handle)** <email or contact>
  - Brief description of contribution
```
